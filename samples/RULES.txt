# 默认规则说明
# ==============================================
# 客户名别名（在.irec_config.json中配置）：
#   canonical -> [alias1, alias2, ...]
#
# 示例：
#   华为技术有限公司 -> ["华为"]
#   阿里巴巴集团 -> ["阿里", "Alibaba"]
#   腾讯科技 -> ["腾讯"]
#   字节跳动 -> ["字节", "ByteDance"]
#   百度在线 -> ["百度"]
#   京东集团 -> ["京东"]
#   美团点评 -> ["美团"]
#   小米科技 -> ["小米"]
#
# 金额容差默认: ±0.01 元
# 日期容差默认: ±3 天
# 匹配优先级:
#   规则1: 一对一（客户名+金额+日期）
#   规则2: 多对一（同客户多发票合并匹配流水）
#   规则3: 一对多（同客户多流水合并匹配发票）
#
# 注意：超出容差的一对多关系保持未解决（不匹配）
